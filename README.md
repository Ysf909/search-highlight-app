# Search Highlight App

## Overview

This is a simple frontend application that allows users to search through a list of articles. As the user types in the search box, matching results are displayed and the matched text is highlighted.

## Features

* Real-time search filtering
* Highlight matching keywords
* Displays number of results found
* Handles empty results case

## Technologies Used

* HTML
* CSS
* JavaScript (Vanilla)

## How to Run the App

1. Download or clone the project files
2. Make sure all files are in the same folder:

   * index.html
   * style.css
   * script.js
3. Open `index.html` in your browser

That's it — no installation or backend required.

## Notes

* The app uses a static array of articles (no backend needed)
* Search is case-insensitive
* Highlighting is done using simple regex

## Possible Improvements

* Add debounce for better performance
* Use a framework like React for scalability
* Add support for searching descriptions (not just titles)
